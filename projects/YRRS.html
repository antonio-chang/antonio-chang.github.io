Yelp Restaurant Recommendation System

Introduction - Yelp Restaurant Recommendation System

• Designed a web application that enables user to search desired a restaurant based on relevant reviews.

• Utilized Yelp Fusion API and a retrieval model with Dirichlet Prior Smoothing to rank top 10 search results by indexing over 6 GB Yelp review data set


  • React
  • Spring boot
  • Dirichlet Prior Smoothing
